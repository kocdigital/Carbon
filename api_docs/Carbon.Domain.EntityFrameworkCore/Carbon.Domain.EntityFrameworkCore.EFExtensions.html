    where T : IHaveOwnership<U>, IEntity where U : EntitySolutionRelation
    Parameters
    Type Name Description
    System.Linq.IQueryable<EFExtensions.RelationEntityPair<T, U>> relationEntities

    The IQueryable of relation entities containing pairs of entities of type T and relations of type U.
    System.Collections.Generic.List<System.Guid> filter

    A list of Guids used to filter the relations based on SolutionId, can be null or empty to include all relations.
    System.Collections.Generic.IList<Orderable> orderables

    A list of Orderable objects specifying the ordering of the entities.
    System.Int32 pageIndex

    The index of the page to retrieve (starting from 0).
    System.Int32 pageSize

    The maximum number of entities per page.
    Returns
    Type Description
    PagedList<T>

    A PagedList containing the requested page of entities with filtered relations, based on the specified criteria.
    Type Parameters
    Name Description
    T

    The type of entities that have ownership relationships with type U and implement IEntity and IHaveOwnership interfaces.
    U

    The type of entity relations that are associated with type T and implement EntitySolutionRelation interface.
    Remarks

    This method retrieves a paged list of entities of type T that have ownership relationships with type U. The method filters the relations based on the provided list of Guids, which represent SolutionIds, if the filter list is not null or empty. The resulting entities are ordered based on the specified ordering criteria and returned as a paged list.

    Remarks

    This method performs a case-insensitive search. If searchByWords is false, it performs a normal contains search for the entire search term. If searchByWords is true, it splits the search term into words and checks if any word exists in the specified property's string value.
